IBM Tech Task by Martin Majorov (2020.11.20)

Java Spring App

Requirements

Create a rest api service which reads the data from this rest api endpoint: https://www.vrk.lt/statiniai/puslapiai/rinkimai/rt.json

should return the election date, election name and election id. It should be possible to do the sorting of data by date by passing the parameter to the rest api.

Implementation:

This solution implements Java Spring Framework to acquire the date from the mentioned API.

The acquired data is filtered and sent to controller in JSON format.

The solution can be found by entering /results endpoint.

/results endpoint sends back unsorted data.

To sort data you need to enter sorted parameter with value of 1 (/results?sorted=1)

If sorted=0 or not entered, /result endpoint always sends back unsorted JSON response.
